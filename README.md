# First Carolina

**We Are the Exception.**

Official website for First Carolina — a North Carolina social media collective and movement focused on connecting young Republicans, activists, and rising conservative voices. Promoting one another. Driving real intra-party debate. Pushing the best young Republicans forward.

## Live Site

Once GitHub Pages is enabled:  
`https://rylanarmstrongnc-droid.github.io/first-carolina/`

## Pages

- **Home** – Hero + Mission preview + paths
- **About** – Full mission
- **Get Involved** – Application form (debate / amplify / join)
- **Contact** – Direct phone, email, Instagram
- **Debates** – Placeholder for the intra-party debate system

## Form Setup

The application form uses Formspree.  
To activate it:

1. Create a free account at [formspree.io](https://formspree.io)
2. Create a new form
3. Replace the `action` URL in `involved.html` with your Formspree endpoint

## Updating the Site

You have full control. Edit files directly on GitHub or ask for changes and they can be pushed for you.
